Current Project




This project is a fullstack application (React, Node, Express, MongoDB) that signs up customers for a subscription payment. The contact form sends an email to the agents email. It also has an agent dashboard - the link to that can be found at the footer of the site.

The agent can create, edit and delete the customers details (CRUD operations). They can also get data about all customers and more info about individual customers. This is a freelance project I am working on most of it is finished, I just have to make a few tweaks to the design and functionality. This is however the first of the two sites I have been asked to build, the second is a more complex booking ecommerce site. To learn more about the second project feel free to contact me.
